Bankruptcy Predictions¶

Running a business is a complex thing, especially on the finacial side. So, if a business goes into bankruptcy it can be often hard to figure out what went wrong. This means that it can be hard for companies to see the signs of bankruptcy before it's too late. Therefore I will be using the Company Bankruptcy Predictions data set in order to create a model that predicts whether or not a company goes bankrupt before signs become obvious.

Conclusion¶

So while we are able to create a model with good accuracy using naive bayes classification. Logistic classification gave us a model that was no more accurate than a coin flip. However, using logistic classification wasn't the wrong thing to do. We have to test multiple different ways to build models in order to figure out which ways works the best for our particular problem. In this case of course, naive bayes classification ended up being the best way to build this model.
